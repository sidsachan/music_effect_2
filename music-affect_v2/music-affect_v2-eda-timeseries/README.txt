Each xlsx file corresponds to physiological signals of all participants listening to one song 

Column A is the timeseries info

Column B onwards are participants data, each column represents a participant.

S1, S2, S3, S4 = classical; S5,S6, S7, S8 = instrumental; S9, S10, S11, S12 = pop

You can use the same label info as music-features.xlsx to do predictions.
